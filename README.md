# Money Counter

A simple web application to track money earned based on hourly rate and time worked.

## Features
- Input your hourly rate (€/hour).
- Track time worked with a start/pause/continue timer.
- View money earned in real-time.
- Save progress locally using browser storage.
- Stop the timer and add elapsed time to "Hours so far."

## How to Use
1. Enter your hourly rate in the "€ / hour" field.
2. Enter any initial hours worked in the "Hours so far (hh:mm)" field.
3. Click "Continue" to start the timer.
4. Click "Pause" to pause the timer.
5. Click "Stop" to add the elapsed time to "Hours so far" and reset the timer.
6. The money earned is displayed in real-time and also shown in the browser tab title.
7. Optionally add a goal you're working towards.
